This repository provides COVID-19 vaccination statistics in machine-readable format. You can learn more in the repository readme.

Documentation for this data will be improved in due course. In the meantime:

To access the data programatically, access the index JSON file at https://vaccinedata.covid19nearme.com.au/data/publications.json. This will list all the PDF files published by the Department of Health. Entries with a vaccineDataPath entry have been parsed correctly and contain vaccination statistics which can be accessed via the provided URL.
